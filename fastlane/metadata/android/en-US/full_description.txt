<p><i>Ankiconnect Android</i> allows you to utilize the standard Anki mining workflow on Android devices like phones and eReaders. Create Anki cards using <a href='https://foosoft.net/projects/yomichan/' target='_blank' rel='nofollow noopener'>Yomichan</a> on <a href='https://kiwibrowser.com/' target='_blank' rel='nofollow noopener'>Kiwi Browser</a> and add them straight into your Anki deck! Mine on the go in the same way as you mine on your desktop pc. Forvo audio is now supported!</p><p><i>Ankiconnect Android</i> is a from scratch unofficial reimplementation of the <a href='https://github.com/FooSoft/anki-connect' target='_blank' rel='nofollow noopener'>desktop Ankiconnect extension</a> and <a href='https://github.com/jamesnicolas/yomichan-forvo-server' target='_blank' rel='nofollow noopener'>desktop Forvo Server extension</a>. It reimplements the core APIs used by Yomichan to work with <a href='https://github.com/ankidroid/Anki-Android/' target='_blank' rel='nofollow noopener'>Ankidroid</a>.</p><p>For instructions, please see the project's Readme.</p>
